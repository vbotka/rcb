RCB (Rsync-Crypto-Backup)
=========================

This is an open-source `BSD-2-clause license
<https://opensource.org/license/bsd-2-clause/>`_ light-weight remote `incremental backup <https://en.wikipedia.org/wiki/Incremental_backup>`_ using `rsnapshot <https://rsnapshot.org/>`_, `rsyncrypto <https://rsyncrypto.lingnu.com/>`_, and `rsync <https://rsync.samba.org/>`_. Feel free to `share your feedback and report issues
<https://github.com/vbotka/rcb/issues>`_.

`Contributions are welcome <https://github.com/firstcontributions/first-contributions>`_.

source: https://github.com/vbotka/rcb
