RCB (Rsync-Crypto-Backup)
=========================

This is an open-source BSD-2-clause light-weight incremental backup
solution using *rsnapshot, rsyncrypto, and rsync*. Feel free to `share
your feedback and report issues
<https://github.com/vbotka/rcb/issues>`_. `Contributions are welcome
<https://github.com/firstcontributions/first-contributions>`_.

source: https://github.com/vbotka/rcb
